# World Wildlife Information & User Management System

A dynamic, data-driven web encyclopedia dedicated to showcasing global animal categories (such as predators, mammals, and domesticated pets) alongside an administrative management portal. 

## 📜 Project Background & Journey
* **Early Academic Project:** This application was designed and developed during my **10th grade** computer science studies (as part of my 11-unit software engineering specialization). 
* **Significance:** It represents my early foundational steps into Full-Stack web development, demonstrating self-taught troubleshooting and an early grasp of relational data and application logic.

## 🚀 Key Features

* **Global Wildlife Knowledge Base:** A structured information system detailing various animal species worldwide, categorized by types (mammals, predators, pets).
* **Media Gallery:** An interactive image gallery showcasing wildlife from around the globe.
* **User Authentication:** Registration and Login systems to separate public visitors from authorized users.
* **Role-Based Administrative Portal:** 
  * **Regular Users:** Can browse the homepage, access the animal knowledge base, and view the gallery.
  * **Administrators:** Access a dedicated dashboard to audit registered users, delete accounts, and execute database queries.
* **Dynamic Complex Query Engine:** A custom-built admin search interface allowing users to construct dynamic SQL queries using conditional logic (`AND` / `OR`) across multiple user attributes (e.g., city, birth year, hobbies, gender).

## 🛠️ Tech Stack

* **Backend / Server-Side:** C# (.NET Framework, ASP.NET Web Forms - `.aspx`)
* **Database:** SQL (Relational queries and data binding)
* **Frontend / Client-Side:** HTML5, CSS3, JavaScript (Dynamic DOM manipulation for conditional search fields)
